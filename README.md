# SiagaBot — Asisten Darurat Bencana Berbasis AI (Offline-First) 

**SiagaBot** adalah aplikasi web responsif yang dirancang sebagai asisten darurat mandiri untuk memberikan panduan keselamatan cepat saat terjadi bencana alam di Indonesia. Proyek ini dikembangkan untuk **IDCamp Challenge 2026**.

### 🌟 Fitur Utama
* **Protokol Resmi:** Panduan evakuasi dan keselamatan disusun berdasarkan standar **BNPB**, **Basarnas**, dan **BMKG**.
* **Offline Capability:** Dirancang dengan arsitektur ringan (Single HTML File) yang memungkinkan akses cepat tanpa ketergantungan penuh pada koneksi internet yang stabil saat darurat.
* **Intelligent Intent Matching:** Menggunakan logika pemrosesan bahasa sederhana untuk mengenali kebutuhan pengguna (Banjir, Gempa, Kebakaran, P3K, dll).
* **Quick Scenarios:** Akses cepat satu klik untuk berbagai jenis bencana utama di Indonesia.
* **Direktori Nomor Darurat:** Daftar lengkap nomor telepon penting (117, 115, 119, dll) yang siap digunakan.

### 🛠️ Teknologi yang Digunakan
* **HTML5 & CSS3:** Menggunakan CSS Grid/Flexbox untuk layout modern dan responsif (Mobile-First).
* **Vanilla JavaScript:** Logika pencocokan kata kunci (NLP sederhana) dan manajemen state aplikasi tanpa library eksternal untuk performa maksimal.
* **Google Fonts:** Menggunakan *Plus Jakarta Sans* untuk keterbacaan yang optimal.

### 📂 Struktur Pengetahuan (Knowledge Base)
Aplikasi mencakup panduan untuk:
1.  **Banjir:** Evakuasi, persiapan, dan tindakan pasca-banjir.
2.  **Gempa Bumi:** Protokol *Drop-Cover-Hold On*.
3.  **Tsunami:** Peringatan dini dan jalur evakuasi dataran tinggi.
4.  **Kebakaran:** Penggunaan APAR (metode PASS) dan prosedur RACE.
5.  **P3K Dasar:** Penanganan luka, patah tulang, dan tersedak (Heimlich Maneuver).
